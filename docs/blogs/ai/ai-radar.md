# 搭建 AI 情报推送系统

## 为什么做这个

AI 行业发展太快，每天都有新模型、新工具、新论文发布。手动跟踪太累，需要一个自动化系统来帮我筛选和推送。

## 系统架构

```
RSS 源 → 内容抓取 → 去重 → LLM 摘要 → 格式化 → 推送
```

### RSS 订阅
精选了以下类型的 RSS 源：
- 头部 AI 公司博客（OpenAI、Anthropic、Google AI）
- 学术论文（arXiv AI 分类）
- 技术社区（Hacker News、Reddit ML）
- 中文 AI 资讯

### DeepSeek 处理
抓取到的文章经过 DeepSeek 处理：
1. **摘要** — 提取文章核心观点
2. **分类** — 按技术领域自动归类
3. **评分** — 评估内容质量，过滤低质内容

### 多渠道推送

#### 飞书机器人
通过飞书群机器人 Webhook 推送，支持富文本格式。

#### Telegram
通过 Bot API 推送到 TG 频道，支持 Markdown 格式。

## 部署

使用 GitHub Actions 定时触发，无需服务器，完全免费。

## 效果

每天早上自动推送精选 AI 资讯，包含原文链接和 AI 摘要。不再错过重要信息。

## 需求来源

洛雪音乐是一款优秀的开源音乐播放器，但导入网易云歌单一直不方便。每次换电脑或重装系统，几百首收藏的歌曲都要手动重新添加。

## 技术分析

### 网易云歌单 API
网易云音乐的歌单数据通过 API 获取，关键步骤：
1. 解析歌单分享链接，提取歌单 ID
2. 调用歌单详情 API 获取歌曲列表
3. 解析返回的加密数据

### Cookie 处理
网易云的 API 需要登录态，方案对比：
- **账号密码登录** — 需要处理验证码，复杂度高
- **Cookie 复用** — 读取浏览器已有的 Cookie，简单安全

选择了 Cookie 复用方案，用户在本地浏览器登录后，工具自动读取 Cookie。

## 安全设计

```
用户浏览器 → [本地读取] Cookie → [仅内存使用] → 获取歌单 → 完成即销毁
```

- Cookie 不上传、不存储、不记录
- 所有网络请求可审查
- 工具完全开源

## 导入流程

```
1. 复制网易云歌单链接
2. 工具自动识别歌单 ID
3. 获取歌单歌曲列表
4. 转换为洛雪音乐格式
5. 一键导入
```

## 遇到的技术挑战

### 歌词编码
不同平台的歌词编码不一致，需要自动检测和转换。

### 歌曲匹配
网易云和洛雪音乐的歌曲 ID 体系不同，需要建立映射关系。

### 批量处理
大歌单（500+ 首）的批量处理性能优化。

## 灵感来源

看到文字像水面一样泛起涟漪的效果，决定自己实现一个。

## 物理原理

### 2D 离散波动方程

波纹传播的核心是波动方程。在离散系统中，每个像素点的波动可以用以下公式描述：

```
U(t+1)[i][j] = 2(1 - a²) · U(t)[i][j] - U(t-1)[i][j] + a² · (U左 + U右 + U上 + U下)
```

其中 `a` 是波的传播速度，`U(t)` 是 t 时刻的高度场。

### 阻尼
为了让波纹逐渐消失，每个时间步对幅值施加阻尼系数：
```
U *= (1 - damping)
```

### 交互响应
鼠标位置作为扰动源，在点击或划过时给对应位置施加一个脉冲：
```
U(鼠标位置) = pulse_force
```

## 渲染实现

### Canvas 渲染
使用 Canvas 2D API，将高度场映射为像素颜色：
- 波峰显示为亮色
- 波谷显示为暗色
- 平静区域显示为原始文字

### 性能优化
- 双缓冲避免闪烁
- requestAnimationFrame 驱动渲染循环
- 只更新变化区域

## 最终效果

鼠标划过文字，涟漪自然扩散，文字像水面一样波动。

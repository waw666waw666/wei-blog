## 项目目标

在 Windows 11 上实现 macOS 风格的提醒应用，做到"看起来像 Mac，用起来是 Win"。

## 设计复刻

### 毛玻璃效果
Windows 上实现毛玻璃有两种方案：
- **Win32 API** — 性能好但兼容性差
- **自绘制** — 灵活但工作量更大

选择了自绘制方案，用 GDI+ 实现类似 macOS 的模糊背景。

### 圆角与阴影
macOS 的圆角体系非常细腻，每个控件的圆角都经过精心设计。在 Windows 上使用自绘控件来还原这种质感。

### 字体渲染
Windows 的字体渲染和 macOS 差异很大。通过调整 ClearType 参数和使用 Web Font，尽量接近 macOS 的字体观感。

## 功能实现

### 通知系统
使用 Windows 原生通知 API（`winrt`），与应用系统深度集成。

### 开机自启
通过注册表实现开机自启，同时提供开关选项，尊重用户选择。

### 单文件部署
使用 PyInstaller 打包为单文件，不写注册表（除开机自启选项外），做到绿色便携。

## 技术踩坑

### 高 DPI 适配
在 4K 屏幕和普通屏幕上都要有好的显示效果，需要处理 DPI 感知和缩放。

### 深色模式
Windows 的深色模式支持不如 macOS 完善，需要手动适配。

## 成果

一个 5MB 不到的 exe 文件，在 Windows 上跑出了 macOS 的质感。

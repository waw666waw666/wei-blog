## 项目背景

claude-cmd-tutor 是一个交互式 CLI 学习平台，帮助开发者快速掌握 Claude Code 的 118 个命令。核心需求是让 CLI 学习像玩游戏一样有趣。

## 技术选型

### 为什么选 TypeScript？
- 类型安全对 CLI 命令解析这种强逻辑场景至关重要
- 丰富的 Node.js 生态支持
- 前端和后端逻辑可以共用类型定义

### 终端模拟
为了在 Web 环境中还原真实的终端体验，选择了以下方案：

- **输入处理**: 自定义命令解析器，支持参数补全和错误提示
- **输出渲染**: 模拟终端输出格式，包括颜色、进度条、表格等
- **异步支持**: 模拟长时间运行的命令，展示实时输出效果

## 架构设计

```
命令解析器 → 命令执行引擎 → 输出格式化 → 终端渲染
     ↑              ↑              ↑
  命令注册表   沙箱环境      主题系统
```

### 命令分类
118 个命令被分为以下几个大类：
- 文件操作
- 代码编辑
- 搜索导航
- Git 集成
- 测试调试
- 项目管理

### 成就系统
内置成就引擎，通过监听用户行为触发成就解锁：
- "初次运行" — 运行第一个命令
- "十项全能" — 掌握 10 个不同分类的命令
- "百晓生" — 学习超过 100 个命令

## 开发过程

最大的挑战是如何在 CLI 教学和真实体验之间找到平衡。太真实会让学习曲线陡峭，太简化又失去了实用性。最终选择了"渐进式难度"的设计：从 guided 模式开始，逐步过渡到 free 模式。

## 项目精选

### [claude-cmd-tutor](https://github.com/waw666waw666/claude-cmd-tutor)
交互式 CLI 学习平台。涵盖 Claude Code 118 个命令的模拟终端教学系统，内置成就引擎，渐进式难度设计。

### [SettingsNav](https://github.com/waw666waw666/SettingsNav)
Windows 设置快速启动器。类似 macOS Spotlight 的搜索体验，支持模糊拼音匹配、多语言、高 DPI 适配。

### [ToDo 极简待办](https://github.com/waw666waw666/ToDo-Minimalist-daiban)
3MB 的极致待办应用。零外部依赖，拖拽排序，本地持久化，追求在功能丰富和极致简洁之间的平衡。

### [Win11 桌面提醒管家](https://github.com/waw666waw666/Win11-Reminder-tongzhi)
在 Windows 上复刻 macOS 美学。5MB 单文件，毛玻璃效果，圆角体系，深色模式适配。

### [AI 情报推送系统](https://github.com/waw666waw666/ai-tg-radar-bot)
RSS + DeepSeek + 飞书/Telegram 全自动 AI 资讯推送。7x24 自动运行，AI 摘要过滤。

### [AI HOT 情报流](https://github.com/waw666waw666/ai-hot-rss-feishu)
极简 Node.js 情报推送。RSS → GitHub Actions → 飞书群，去重缓存 git 化，零服务器成本。

### [前端组件平台](https://github.com/waw666waw666/frontend-components-skill)
51+ 组件的展示平台。React + TypeScript，按需加载，代码示例一键复制，AI 模板生成。

### [文字水波纹 Canvas 效果](https://github.com/waw666waw666/pretext-ripple)
2D 离散波动方程的 Canvas 实现。物理模拟 + 60fps 渲染，鼠标交互产生涟漪扩散。

### [lx-music-import](https://github.com/waw666waw666/lx-music-import)
歌单迁移工具。网易云到洛雪音乐的一键导入，Cookie 复用方案，安全设计。

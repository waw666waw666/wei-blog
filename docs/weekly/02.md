## 独立开发

- 做小工具不需要完美。先把 MVP 发出来，有人用再迭代
- 一个 3MB 的待办应用比一个 30MB 的"全能应用"更有竞争力——前提是它真的把一件事做好了
- GitHub Actions 是免费服务器的最佳替代品。定时任务、自动化推送、数据采集，都能跑

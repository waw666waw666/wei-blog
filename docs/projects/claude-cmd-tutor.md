# claude-cmd-tutor

**互动式 Claude Code 命令学习平台**

GitHub：[waw666waw666/claude-cmd-tutor](https://github.com/waw666waw666/claude-cmd-tutor)

一个交互式 CLI 学习平台，帮助你快速掌握 Claude Code 的 118 个命令。包含交互终端模拟、情景挑战和成就系统三大模块，让 CLI 学习像玩游戏一样有趣。

## 技术栈

TypeScript、Node.js

## 亮点

- 118 个命令全覆盖，分类学习
- 内置交互终端，边学边练
- 情景挑战模式，真实场景演练
- 成就系统，记录学习进度

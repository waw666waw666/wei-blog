## 项目简介

基于 2D 离散波动方程的 Canvas 动画，文字像水面一样泛起涟漪，鼠标划过产生真实的物理波动效果。

## 技术栈

TypeScript、Canvas

## 核心原理

### 波动方程
使用 2D 离散波动方程模拟水面波纹传播：
```
U(t+1) = 2(1 - a²)U(t) - U(t-1) + a²(U左 + U右 + U上 + U下)
```

### 文字与波的交互
将文字像素映射为波源位置，鼠标移动产生扰动源，波纹在文字表面自然传播。

### 渲染优化
使用 Canvas 逐帧渲染，通过双缓冲技术避免画面闪烁，配合 requestAnimationFrame 实现 60fps 流畅动画。

## 效果

鼠标划过文字时，文字像水面一样泛起涟漪，产生真实的物理波动效果，交互反馈即时且自然。

## 项目简介

像素级复刻 macOS 美学设计，免安装单文件运行，极致轻量 + 托盘常驻，使用 Windows 原生通知。

## 技术栈

Python

## 设计特色

### 仿 macOS 美学
从毛玻璃效果到圆角设计，在 Windows 上还原 macOS 的视觉体验。

### 单文件运行
无需安装，下载即用，不写注册表，不留垃圾文件。

### 原生通知集成
使用 Windows 11 原生通知系统，与系统融为一体。

## 功能亮点

- 设置定时提醒
- 重复提醒（每天/每周/自定义）
- 系统托盘常驻
- 开机自启
- 轻量级，内存占用极低

## 项目简介

点击即可跳转到 Windows 系统设置的对应位置。支持搜索、收藏常用设置、添加 EXE 快捷方式、托盘常驻，以及中英文界面切换。

## 技术栈

- **语言**: TypeScript
- **平台**: Windows (Win32 API)

## 核心功能

### 智能搜索
输入关键词即可快速定位到对应的 Windows 设置页面，支持模糊匹配和拼音搜索。

### 收藏管理
将常用设置添加到收藏夹，支持自定义分组和排序。

### EXE 快捷方式
不仅支持系统设置，还可以添加任意 EXE 程序的快捷方式，打造个性化启动器。

### 托盘常驻
最小化到系统托盘，通过快捷键或鼠标点击随时呼出，不干扰工作流。

### 国际化
完整支持中英文双语界面，自动检测系统语言。

## 技术难点

调用 Windows 设置 URI (`ms-settings:`) 时，不同 Windows 版本支持的 URI 方案存在差异。通过维护一份兼容性映射表，在不同系统版本上自动适配。
